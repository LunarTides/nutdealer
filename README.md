# NUTDEALER

So this is basically a Deltarune dark world creator. It is very WIP, so expect things to break constantly.

I have some ideas on what features to implement and stuff, but I'm keeping those for myself for now since it's in flux. The [issues](https://git.lunartides.dev/LunarTides/nutdealer/issues) only contains smaller features/bugfixes for now.

---
Here are some fundementals. There are still some bugs / small features that haven't been implemented yet.
- [x] Title Screen
- [x] Dark World Creator
    - [x] Camera that can pan
    - [ ] Camera that can zoom
    - [x] Grid
    - [x] Place tiles on said grid
        - [x] Give them different textures
        - [x] Delete, copy, move them around the grid
        - [x] Change their behaviour (solid, hide during play, etc...)
        - [x] Give them logic using GDScript scripts (run code when interacted with, etc...)
            - [x] Save/load scripts
    - [x] Save/load dark worlds
    - [x] Preview
- [x] Play dark worlds
